# nasa-wildfires-chart
Showing Wildfires With Nasa's EONET API

## Usage

### Run Frontend

```
$ npm install
$ npm run start
```
### Run Tests

```
$ npm run test
```
To get more details of the API - Please follow this link <a href="https://eonet.sci.gsfc.nasa.gov/docs/v3">https://eonet.sci.gsfc.nasa.gov/docs/v3</a>
